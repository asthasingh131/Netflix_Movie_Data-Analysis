# Netflix_Movie_Data-Analysis

## Overview
This project demonstrates an end-to-end data analytics workflow using Python to analyze movie data and uncover insights
related to genres, popularity trends, and yearly movie production. The findings are summarized and communicated through 
a professional presentation.

---

## Dataset
- Structured movie dataset containing titles, genres, popularity scores, vote counts, and release years.
- Cleaned and prepared using Python before analysis.
- Used for exploratory and trend-based analysis.

---

## Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn) – EDA & data cleaning  
- **Jupyter Notebook** – Analysis workflow  
- **Gamma** – Business presentation (PPT)

---

## Project Steps
1. Load the dataset in Python  
2. Perform exploratory data analysis (EDA)  
3. Clean and preprocess the data  
4. Visualize trends and patterns  
5. Extract key insights  
6. Create a presentation using Gamma  

---

## Presentation
- Genre distribution and popularity trends  
- Top-performing movies  
- Year-wise movie production analysis  
- Insights presented in a clear, non-technical format  

---

## Results
- Drama identified as the most dominant and popular genre  
- Blockbuster movies show exceptionally high popularity scores  
- 2020 recorded the highest number of movie releases  
- Insights support understanding of audience preferences and industry trends  

---

## How to Run
1. Clone the repository  
2. Install required Python libraries  
3. Run the Jupyter Notebook to reproduce the analysis
